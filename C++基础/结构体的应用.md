### 结构体的应用
* 结构体可以整体进行交换，不必像下面例子一步步地进行
```C++
    struct person tmp;
    tmp.age = s[j].age;
    tmp.name = s[j].name;
    s[j].age = s[j+1].age;
    s[j].name = s[j+1].name;
    s[j+1].age = tmp.age;
    s[j+1].name = tmp.name;
    flag = true;
```
只需要
```C++
    struct person tmp = s[j];
    s[j] = s[j+1];
    s[j+1] = tmp;
```

* 书写时注意下标，`i` `j`不要弄混了

### 冒泡排序优化bool变量的位置
* 位置一
```C++
    for(i=0;i<num;i++)
    {
        bool flag = false;
        for(j=0;j<num-i-1;j++)
        。。。
        ...
    }
```
* 位置二
```C++
    bool flag = false;
    for(i=0;i<num;i++)
    {
        for(j=0;j<num-i-1;j++)
        。。。
        ...
    }
```
位置二是**错误**的，若发生了交换，进行下一次循环的时候，`bool`变量未被还原成`false`,即使下一次未发生交换，变量的值仍是`true`,达不到优化的目的。
